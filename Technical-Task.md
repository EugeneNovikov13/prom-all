Задача: Реализовать приложение «Сайт Промышленный Альянс».

Страницы:

- Главная;
- Каталог;
- Документы;
- О нас;
- Контакты;
- Авторизация администратора;
- Админ-панель;
- Страница с ошибкой о том, что страница по заданному адресу не существует;
- Страница с ошибкой о том, что у пользователя нет доступа к заданной странице.

Пояснения:

- Всего у пользователя может быть 2 роли: Администратор, Гость;
- Если какая-то страница не доступна для пользователя, то она не должна открыться при переходе по прямой ссылке. В таком случае должна открыться страница с ошибкой о том, что доступ запрещен;
- Картинки для статей — удаленные адреса из файлохранилища (ссылка);

Технические требования:

- Фронт-энд на базе Create React App;
- Обеспечение качества кода: ESLint, Prettier, EditorConfig;
- CSS: Styled Components;
- Верстка под Desktop и Mobile;
- Типизация отсутствует на первом этапе;
- Формы с валидацией: React Hook Form и Yup;
- Сетевые запросы: Axios;
- Роутинг: React Router;
- Менеджер состояния: React Redux, Redux Toolkit;
- Бек-энд - Express на базе Node.js;
- База данных MongoDB, взаимодействие через mongoose;
- Аутентификация пользователя с помощью JsonWebToken;
- Шифрование с помощью bcrypt;
- Контейнеризация на сервере: Docker и Docker-compose;
- Допускается обоснованное использование npm-пакетов.

0. Все страницы

Содержание (3 секции сверху вниз):

- шапка

0.1. Шапка

Содержание:

- Лого и название организации;
- Время работы;
- Электронная почта;
- Контактный телефон;
- Навигационное меню снизу:
  - Главная (ссылка на страницу);
  - Каталог (ссылка на страницу);
  - Информация/Документы (ссылка на страницу);
  - О нас/Бренды (ссылка на страницу);
  - Контакты (ссылка на страницу);
  - Поиск (поиск товаров по наименованию);
- Значок администратора (если администратор авторизован, иначе скрыть);
- Кнопка «Выйти» (если администратора авторизован, иначе скрыть);

Поведение:

- После нажатия на лого или название организации происходит переход на Главную страницу;
- После нажатия на «Электронную почту» всплывает модальное окно с блоком «Быстрая заявка»;
- После нажатия на «Контактный номер» происходит переход на страницу Контакты;
- После нажатия на кнопку «Выйти» меняется роль пользователя на Гость. Происходит переход на Главную страницу;
- После нажатия на кнопки в навигационном меню происходит переход на соответствующие страницы;
- После нажатия на кнопку «Поиск» открывается поле для ввода;
- При вводе текста в поле поиска и нажатии кнопки «Поиск» происходит переход на страницу каталога со всеми товарами, удовлетворяющими условию поиска;
  - После ввода текста и простоя 2 секунды должен автоматически активироваться поиск;
  - После активации поиска отправляется запрос на сервер для обновления выведенного списка товаров;
  - На сервере отбираются товары, в которых есть заданная в поле поиска фраза — совпадение в названии товара;
  - После успешного ответа от сервера отобразить результат на странице «Каталог».

Содержание:
- Блок «Быстрая заявка»

Поведение:
- Всё остальное пространство вокруг содержания модального окна затемняется;
- При нажатии за пределы содержания модальное окно закрывается;
- Всё, что находится вне содержания модального окна становится недоступным для управления.

0.4.1 Блок «Быстрая заявка»

Содержание:

- Форма для отправки на электронную почту быстрой заявки:
  - ФИО;
  - Электронная почта для обратной связи;
  - Телефонный номер для обратной связи;
  - Текст заявки;
- Кнопка «Отправить сообщение».

Поведение:

- После ввода данных в форму и нажатию на кнопку «Отправить сообщение» происходит валидация формы;
- В случае, если валидация не прошла, отображается ошибка;
- Если ошибок нет и валидация пройдена, открывается captcha для блокировки спам-рассылки;
- После успешного прохождения captcha, отправляется электронное письмо с данными из формы.

1. Главная

Адрес: /

Содержание:

- Рекламный блок;
- Блок «Каталог товаров»;
- Блок «Преимущества»;
- Блок «Бренды».

1.1. Рекламный блок

Содержание:

- Наименование товара;
- Текст объявления;
- Контактный номер;
- Кнопка «Сделать заказ»;
- Набор кнопок перехода по товарам.

Поведение:

- Фоновое изображение приближенно соответствует категории предлагаемого товара;
- Товары сменяют друг друга по прошествии заданного интервала;
- При нажатии на кнопку «Сделать заказ» открывается модальное окно «Быстрая заявка» с предварительно заполненными данными заказа;
- При нажатии на кнопку перехода по товарам слайдер переключается на соответствующий товар.

1.2. Блок «Каталог товаров»

Содержание:

- Карточки основных категорий товаров.

Поведение:

- При нажатии на любое место карточки открывается список подкатегорий;
- При нажатии на блок, другие блоки закрываются;
- При нажатии на кнопку подкатегории происходит переход на страницу «Каталог» в соответствующую подкатегорию товаров;
- Если подкатегорий больше 5, то снизу списка появляется кнопка «Все товары»;
- При нажатии на кнопку «Все товары», происходит переход на страницу «Каталог» в соответствующую категорию товаров;
- При выходе указателя мыши за пределы блока, блок закрывается в исходное состояние.

1.3. Блок «Преимущества»

Содержание:

- Горизонтальный перечень достоинств компании.

Поведение:

- После нажатия на один из блоков перечня раскрывать текст «Преимущества»;
- После нажатия на открытый блок перечня он закрывается;
- Если один из блоков перечня раскрывается, то другие автоматически закрываются.

1.4. Блок «Бренды»

Содержание:

- Бесконечный слайдер, демонстрирующий основные бренды, сотрудничающие с организацией;
- Кнопка «Подробнее».

Поведение:

- Карточки брендов сменяют друг друга по прошествии заданного интервала;
- Слайдер можно плавно двигать влево-вправо;
- После нажатия на кнопку «Подробнее» раскрывается страница «О нас/Бренды».

2. Каталог

Адрес: /products

Содержание:

- Breadcrumbs;
- Карточки основных категорий товаров;
- Навигационное меню слева (?);
- Блок карточек подкатегорий/разделов/товаров;
- Страница товара;
- Блок «Подсказка»

2.1. Breadcrumbs

Содержание:

- Горизонтальное навигационное меню, повторяющее структуру каталога:
  - Каталог (страница «Каталог»);
  - Категория товаров;
  - Подкатегория товаров;
  - Раздел подкатегории (если есть);
  - Товар;
- Кнопки, раскрывающие список категорий/подкатегорий/разделов/товаров, в виде стрелочки.

Поведение:

- При нажатии на Каталог возвращает на страницу «Каталог»;
- При нажатии на стрелочку раскрывается доступный перечень категорий/подкатегорий/разделов/товаров;
- Если один перечень развёрнут, то остальные перечни сворачиваются;
- При нажатии на элемент перечня происходит переход на страницу каталога в соответствующий элемент структуры;
- Рядом с товаром стрелки нет, вместо неё галка;
- Повторяет состояние Навигационного меню слева.

2.2. Карточки основных категорий товаров

Поведение и содержание:

- повторяет Блок «Каталог товаров» с «Главной» страницы.

2.3. Навигационное меню слева (под вопросом)

Содержание:

- Вертикальное меню в виде «Аккордеона», раскрывающееся на соответствующем элементе структуры каталога. (Окончательный вид проработать дизайнеру)

Поведение:

- Не отображается на основной странице «Каталог» и на Странице товара;
- При нажатии на элемент меню происходит переход на страницу каталога в соответствующий элемент структуры;
- Если один из блоков меню раскрыт, то остальные блоки сворачиваются;
- Повторяет состояние Breadcrumbs.

2.4. Блок карточек подкатегорий/разделов/товаров

Адрес: /products/:category

Содержание:

- Блок из карточек с названием и изображением соответствующих подкатегорий/разделов/товаров.

Поведение:

- При нажатии на карточку происходит переход в соответствующий элемент структуры каталога;
- Появляются карточки соответствующие новому элементу структуры каталога;
- Меняется состояние Breadcrumbs и Навигационного меню слева.

2.5. Страница товара

Адрес: /products/:id

Содержание:

- Блок фотографий товара;
- Наименование товара;
- Блок выбора количества;
- Блок выбора модели товара;
- Кнопка «Оформить заказ»;
- Блок «Описание товара»;
- Блок «Технические характеристики».
- Блок «Подсказка» (3 блока с подсказками, как заказать товар через форму «Быстрая заявка»).

Поведение:

- Отображается после выбора конкретного товара;
- Блок фотографий существует в виде слайдера с ручным перелистыванием;
- При клике на кнопку «Оформить заказ» открывается модальное окно с блоком «Быстрая заявка», где в поле текста заявки уже внесены:
  - Наименование товара;
  - Выбранная модель товара;
  - Выбранное количество товара;
- Блоки «Описание товара» и «Технические характеристики» раскрываются по клику на них в виде модального окна.

- Если авторизация не удалась, то сообщить об ошибке (либо неверный пароль, либо логин не существует);
